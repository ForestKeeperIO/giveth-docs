# Discord Introdução e Tutorial

## Migração para o Discord


Mudamos a maior parte do chat da nossa comunidade e da colaboração de nosso contribuidores do [Riot.im](http://riot.im/) (agora Element Chat) para um novo servidor no Discord.

Estamos utilizando uma bridge para a transmissão de mensagens entre Discord, Telegram e o Element. Essa bridge espelha as mensagens de alguns canais escolhidos entre as três aplicações. Se o Discord não for para você, não tenha medo! Você ainda pode interagir com a gente utilizando o Telegram ou Element se preferir.

O Software para fazer a bridge entre as salas é o matterbridge. - Converse com o `@geleeroyale` para saber como a magica acontece!

## Como entrar no servidor da Giveth no Discord

Faça seu login no Discord ou crie uma nova conta - [Discord](https://discord.com/) ou baixe o aplicativo ([iOS](https://apps.apple.com/us/app/discord-talk-chat-hang-out/id985746746), [Android](https://play.google.com/store/apps/details?id=com.discord&hl=en_CA&gl=US)).

Não sabe como usar o Discord? [Confira esse Guia Completo](https://support.discord.com/hc/pt-br/articles/360045138571-Guia-do-Iniciante-para-Discord) ou assista esse vídeo: ([Discord - Principiante a Avançado](https://www.youtube.com/watch?v=tNUq5Aqv60s))

Junte-se ao servidor da comunidade Giveth - [discord.giveth.io](https://discord.giveth.io/)

Dentro de cada servidor, você pode ter um apelido único, por isso fique à vontade para escolher como você quer ser reconhecido.

Assim que você entrar no servidor, um bot irá lhe enviar uma mensagem com um Captcha para ser resolvido - você não é um robô, certo? Deve ser algo parecido com isto:

### Comandos do Bot

#### PraiseBot

Na Giveth, adoramos reconhecer o valor que cada membro traz para a comunidade e proporcionar gratidão pelos feitos maravilhosos que eles realizam. Os elogios dados são seguidos pelo PraiseBot e utilizados para calcular recompensas para os contribuidores ativos.

Você pode elogiar um membro da comunidade em qualquer canal, digitando **`!praise`** seguido de "@" e o nick do usuário. Não se esqueça de mencionar porque os elogia.

Por exemplo: `!praise @WhyldWanderer por criar um Guia do Discord tão bom 😛`

Uma vez que você envia o seu elogio, emoji ✅ irá aparecer como reação à sua mensagem, mostrando que o bot reconheceu o comando.

Sempre que você receber um elogio, o bot irá te mandar uma mensagem no privado com um link da mensagem onde você foi elogiado.

#### Bot simples para votação
Este bot pode ser utilizado para criar uma votação como essa:

![](https://i.imgur.com/ExLeqzW.png)

Para criar a votação, basta escrever o comando: `/poll "Qual é o melhor bot para votação? "Simple Poll" "R2D2"`

Seguindo esse modelo, você pode criar quantas respostas quiser e os membros da comunidade podem responder reagindo com os emoji correspondentes. `/poll "Pergunta" "Resposta #1" "Resposta #2" "Resposta #3"` ..etc.

#### Bot Chip

O [Bot Chip](https://chipbot.gg/home) pode ser convidado para qualquer canal de voz no servidor. Uma vez que você o convida, qualquer pessoa pode digitar comandos no canal de texto `#chip-playlist` para começar a tocar uma música  organizar as playlists. Aqui está uma lista dos comandos disponíveis:

**`ch!p`** `link ou nome da música` - Faz com que sua música seja tocada ou adicionada à fila; se não estiver tocando outras músicas e ao usar esse comando, o bot entrará na sala tocando a música desejada.
**`ch!queue`** - Exibe a fila de músicas 
**`ch!skip`** - Pula para próxima música
**`ch!clear`** - Remove todas as músicas da fila
**`ch!skipto`** `posição ou nome da música` - Salta para a música desejada
**`ch!lyrics`** - Exibe a letra da música que está tocando
**`ch!pause`** - Reprodução de pausas
**`ch!resume`** - Retoma da reprodução
**`ch!remove`** `posição ou nome da música` - Remove a música da fila
**`ch!stop`** - Desliga o bot do seu canal de voz e limpa a fila
**`ch!shuffle`** - Randomiza a ordem atual das músicas na fila
**`ch!nowplaying`**- Apresenta informação sobre a música que está sendo tocada no momento.

O Bot Chip pode reproduzir música de várias fontes, incluindo Spotify, YouTube, Soundcloud, etc. Para que o Chip bot possa tocar a partir de Spotify basta clicar em Share > Copy Playlist Link > copie isso para o seu chat com a tag `ch!p` e pronto.
